The Challenge

Why General AI Struggles in Petroleum Engineering

No Domain Terminology

Misunderstands "porosity", "kick", "completion", "water cut"

Unknown Data Formats

Can't read LAS, SEG-Y, WITSML, PRODML, DLIS

Missing Safety Culture

Ignores well control, process safety, environmental protection

Jumps to Coding

Skips design, misses requirements, builds wrong thing

Development Workflow

Rigorous Process: Design → Plan → Build → Review

Brainstorm

  • • Clarify intent
  • • Propose options
  • • Get approval
→

Write Plan

  • • 2-5 min tasks
  • • File paths
  • • Verify steps
→

TDD

  • • RED: write test
  • • GREEN: implement
  • • REFACTOR: clean
→

Execute

  • • Subagents/task
  • • Parallel work
  • • Fresh context
→

Review

  • • Spec compliance
  • • Code quality
  • • Evidence check
Key Principle: "Evidence before assertions" — no claiming success without proof
